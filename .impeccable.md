## Design Context

### Users
- **화원 파트너**: 주문 접수/처리, 배달 관리. 매장에서 PC 또는 모바일로 사용
- **본사 관리자**: 화원 관리, 상품 검색, 시스템 운영. 사무실 PC에서 사용
- **일반 고객**: 지사 홈페이지를 통해 꽃배달 서비스 정보 확인, 주문 문의

### Brand Personality
- **신뢰**: 정확한 배달, 안정적인 서비스를 느낄 수 있는 UI
- **따뜻함**: 꽃과 자연을 연상시키는 부드럽고 따뜻한 색감과 톤
- **전문성**: 깔끔하고 체계적인 레이아웃으로 전문 서비스 인상

### Aesthetic Direction
- **톤**: 자연스럽고 따뜻한 모던 디자인. 과하지 않은 세련됨
- **색상**: 기존 그린 브랜드 컬러(#5B7A3D) 유지, 따뜻한 뉴트럴 톤 강화
- **테마**: 라이트 모드 기본, 밝고 깨끗한 배경
- **안티 레퍼런스**: AI 생성 느낌(다크모드+네온, 글래스모피즘, 그라디언트 텍스트, 보라-파랑 그라디언트) 절대 금지
- **레퍼런스**: 꽃집/플로리스트 웹사이트의 자연스러운 따뜻함 + SaaS 관리 도구의 깔끔한 구조

### Design Principles
1. **자연스러운 따뜻함**: 꽃과 자연을 연상시키되 과하지 않게. 미묘한 색상 틴팅과 따뜻한 뉴트럴 사용
2. **명확한 위계**: 정보 계층이 한눈에 파악되는 타이포그래피와 레이아웃
3. **실용 우선**: 장식보다 기능. 모든 시각 요소는 사용자의 작업을 돕는 목적이 있어야 함
4. **일관된 품질**: 컴포넌트 간 시각적 일관성 유지, 디테일에 신경 쓴 마감
5. **인간적 디자인**: AI가 만든 듯한 generic한 패턴 대신, 의도가 느껴지는 디자인 선택
